Navbar

Navbar 1

A navbar with all elements on the left.

Use a single child under div.navbar.

  1. TITLE

  2. Link 1
  3. Link 2
  4. Link 3
  5. Link 4
					
<div class="navbar">
	<ul>
		<li><h1 class="title">TITLE</h1></li>
		<li><a href="#">Link 1</a></li>
		<li><a href="#">Link 2</a></li>
		<li><a href="#">Link 3</a></li>
		<li><a href="#">Link 4</a></li>
	</ul>
</div>

Navbar 2

	</ul>
	
	<ul>

	</ul>
</div>
